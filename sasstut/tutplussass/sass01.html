testing...

        1. touch style.scss
        2. add..
            body {
                height: 200px;
                width: 200px;
                background: #fff;
            }

        3. setup "watch file"
            a. sass --watch style.scss
            b. >>> Sass is watching for changes. Press Ctrl-C to stop.
            c. code... code... code... code... save..
            d. >>> Change detected to: style.scss
                  write style.css
                  write style.css.map

        4. sass --style compressed  style.scss:style.css

            // from:

                // variable
                $main-color: green;

                // nesting
                ul {
                    background: red;
                    li {
                        display: inline;
                    }
                }

                body {
                    height: 200px;
                    width: 200px;
                    background: $main-color;
                }

            // to:
            // style used -- compressed

                ul{background:red}ul li{display:inline}body{height:200px;width:200px;background:green}

        5. Sass Mixin

            // radius with default 5px radius, you can overwrite the (5px) value..
            @mixin rounded (radius : 5px) {
                -webkit-border-radius: $radius;
                -moz-border-radius: $radius;
                border-radius: $radius
            }

                // to use
                .box {
                    // overwrite 5px with 10px..
                    @include rounded (10px);
                }

        6. Selector inheritance

            // man rounded box...
            .roundedBoxMain {
                -webkit-border-radius: 5px;
                -moz-border-radius: 5px;
                border-radius: 5px
            }

            // use the main box elements
            .roundedBox01 {
                @extend .roundedBoxMain;
                bgcolor: red;
            }
                

            // use the main box elements
            .roundedBox02 {
                @extend .roundedBoxMain;
                bgcolor: green;
